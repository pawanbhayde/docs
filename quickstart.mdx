## How to get App ID and App Secret?

1. Go to the [Facebook Developer Console](https://developers.facebook.com/apps) and create a **Business App** for your [Business Manager Account](https://business.facebook.com) if it is not already created.
2. Open the created app.
3. Navigate to **Settings → Basic**.
4. Copy the **App ID** and **App Secret** fields.

## How to get Access Token (Public Access Token)?

1. Go to your [Business Manager Account](https://business.facebook.com).
2. Select your **Business Account / Business Page** from the dropdown.\
   Reference: https://cdn.evenuefy.com/general/documentation/fb_bma_select_ba.png
3. Click the **Settings** button at the bottom.\
   Reference: https://cdn.evenuefy.com/general/documentation/fb_bma_click_setting.png
4. Navigate to **Users → System Users**.
5. Add a **System User** with the required role if not already added.
6. Select the user and click **Add Assets** to assign your App.

Reference Steps:

- https://cdn.evenuefy.com/general/documentation/fb_bma_select_add_assets.png
- https://cdn.evenuefy.com/general/documentation/fb_bma_assign_asset_app.png

7. Click **Generate New Token**.

Permissions required:

- whatsapp_business_messaging
- whatsapp_business_management

8. Click **Generate Token** and copy the generated token.

<Warning>
  The generated token will only be shown once. Save it securely because Facebook will not store it.
</Warning>

Reference Screenshot:\
https://cdn.evenuefy.com/general/documentation/fb_bma_copy_access_token.png

## How to get API Version?

1. Open the [Facebook Developer Console](https://developers.facebook.com/apps).
2. Select your app.
3. Navigate to **Settings → Advanced**.
4. Scroll to **Upgrade API Version**.

Reference:\
https://cdn.evenuefy.com/general/documentation/fb_app_api_version.png

## How to get API URL?

The WhatsApp Business Platform provides multiple APIs.

### Cloud API

Use the Graph API endpoint: https://graph.facebook.com/v14.0

### On-Premises API

Use your **own server domain URL**.

### Business Management API

Used for managing WhatsApp Business accounts, templates, and phone numbers.

For more details see:\
https://developers.facebook.com/docs/whatsapp/overview

## How to Configure Webhook?

1. Open the **Facebook Developer Console**.
2. Setup the **WhatsApp Product** in your app.

Reference:\
https://cdn.evenuefy.com/general/documentation/fb_app_setup_whatsapp.png

3. Navigate to: Products → WhatsApp → Configuration

Reference:\
https://cdn.evenuefy.com/general/documentation/fb_app_wa_webhook_configure.png

4. Configure the following values:

- Callback URL
- Verify Token
- Webhook Fields

5. Click **Edit** to set the **Callback URL** and **Verify Token**.
6. Click **Manage** under Webhook Fields and subscribe to:

messages

https://cdn.evenuefy.com/general/documentation/fb_app_webhook_fields.png

## Supported Media Types

Reference:\
https://developers.facebook.com/docs/whatsapp/cloud-api/reference/media#supported-media-types

| Media    | Supported Types                                                                                                                                                                                                                                                                                                                                 | Size Limit                     |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------ |
| audio    | `audio/aac`, `audio/mp4`, `audio/mpeg`, `audio/amr`, `audio/ogg` (only opus codec supported)                                                                                                                                                                                                                                                    | 16MB                           |
| document | `text/plain`, `application/pdf`, `application/msword`, `application/vnd.ms-excel`, `application/vnd.ms-powerpoint`, `application/vnd.openxmlformats-officedocument.wordprocessingml.document`, `application/vnd.openxmlformats-officedocument.presentationml.presentation`, `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet` | 100MB                          |
| image    | `image/jpeg`, `image/png` (must be 8-bit RGB or RGBA)                                                                                                                                                                                                                                                                                           | 5MB                            |
| video    | `video/mp4`, `video/3gp` (H.264 video codec + AAC audio codec supported)                                                                                                                                                                                                                                                                        | 16MB                           |
| sticker  | `image/webp`                                                                                                                                                                                                                                                                                                                                    | Static: 100KB, Animated: 500KB |

---

<Tip>
  Use media uploads when sending WhatsApp template messages or interactive messages through the Cloud API.
